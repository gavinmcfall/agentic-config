## How to Build Your Own

This skill is a template. The structure below shows you what to fill in.
The original author built theirs using personality frameworks (MBTI, Enneagram,
DISC, neurodivergence profiles). You can use whatever frameworks resonate with you.

The key insight: this skill works because it knows the TENSION between your traits,
not just the labels. If you're an introvert who leads, a perfectionist with ADHD,
a people-pleaser who burns out вЂ” document the contradiction, not just the type.

## How to Populate This Skill

1. Spend time reflecting on your personality frameworks (or take assessments)
2. Document the TENSIONS, not just the types
3. Write down what your "stuck" patterns look like
4. Write down what has actually helped in the past (be specific)
5. Define the response rules вЂ” what kind of support do you need?
6. Test it by invoking when you're actually struggling. Iterate.

See `docs/creating-your-inner-ally.md` for a detailed guide.

## README Generation

The README is auto-generated by `scripts/generate-readme.sh` and includes:

1. **Header**: What this is, who it's for
2. **Quick Start**: `./install.sh` one-liner
3. **What's Included**: Table of skills by tier with descriptions (parsed from SKILL.md frontmatter)
4. **Hooks**: What each hook does
5. **Rules**: What each rule file provides
6. **Customization**: Pointer to docs/
